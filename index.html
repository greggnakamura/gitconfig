gitconfig by codeimpossible

My .gitconfig file

I use git on a lot of different machines and I've been using dropbox to sync my .gitconfig but this can get pretty messy. So I created this repo to host my .gitconfig. Feel free to fork this repo and add to it!

Install

Just clone it into your `~/` directory: 

$ git clone https://github.com/codeimpossible/gitconfig
